# 智能菜谱管理系统 java1212

## 项目概述

智能菜谱管理系统是基于 Spring Boot 和 Vue.js 技术栈开发的，旨在为用户提供便捷的菜谱管理服务。系统前后端分离，具备良好的可扩展性和维护性。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

- 菜谱管理
  - 菜谱新增与编辑
  - 菜谱分类管理
  - 菜谱搜索与筛选
- 用户管理
  - 用户注册与登录
  - 用户权限控制
- 其他功能
  - 数据统计与分析
  - 系统设置与维护

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Java环境：JDK 1.8及以上
- 数据库：MySQL 5.7及以上
- Node.js环境：用于构建前端项目

## 部署步骤

1. 搭建Java开发环境（如安装JDK）
2. 搭建数据库环境（如安装MySQL）
3. 克隆或下载源码到本地
4. 构建前端项目（使用npm或yarn命令）
5. 部署后端项目（如使用Maven构建）
6. 运行后端项目
7. 访问前端项目地址，进行系统测试
